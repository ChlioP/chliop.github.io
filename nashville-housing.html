Nashville Housing Data Cleanup

Data Cleaning and Visualization with SQL & Tableau

Dashboard Preview

This project focuses on cleaning and analyzing historical housing data from Nashville. The dataset had inconsistencies in date formats, duplicate records, and unstructured fields, which I cleaned using SQL. I then visualized key trends using Tableau.

Tools Used

  • MySQL (for data cleaning and transformation)
  • Tableau Public (for visualizing trends)

SQL Tasks Performed

  • Removed duplicates using `ROW_NUMBER()`
  • Formatted dates to YYYY-MM-DD
  • Split owner names into first and last
  • Standardized property addresses

Key Insights

  • Average home prices increased steadily from 2013–2016
  • 3-bedroom homes had the highest sales frequency
  • Certain zip codes consistently had higher property values
